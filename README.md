# 📱 Chat App - Flutter & Firebase



## ✨ Features 

* **Authentication:** تسجيل دخول وإنشاء حساب جديد باستخدام Firebase Auth.
* **Profile Image:** رفع صور الشخصية باستخدام Cloudinary مع ميزة ضغط الصور لضمان السرعة.
* **Private Chat:** نظام شات خاص بين الأصدقاء مع واجهة عصرية (Floating Input Field).
* **Real-time Updates:** استخدام Firestore Streams لتحديث الرسائل والإشعارات لحظياً.
* **Friend System:** إرسال طلبات صداقة، قبولها، أو حذف الأصدقاء.
* **Notifications Badge:** عداد إشعارات ذكي يظهر على أيقونة التنبيهات.
* **Search:** البحث عن مستخدمين آخرين بالاسم.

---

## 🚀 Tech Stack 

* **Frontend:** [Flutter](https://flutter.dev) (Dart)
* **Backend:** [Firebase](https://firebase.google.com) (Firestore & Auth)
* **Image Hosting:** [Cloudinary](https://cloudinary.com)
* **State Management:** StatefulWidget & StreamBuilder
* **Networking:** Http package for Cloudinary API
